# automated_stylesheets
This will be a hybrid subroutine that blends color mixing with a chatbot for plain language stylesheet creation.

## Concept
Retain similar color values and save for mixing later. ( Here only second hex code list is shuffled. )<br />
Mix four retained colors and compress it to one dark color.<br />
Use colors for generating CSS stylesheets. Lightest color is headers and paragraphs. Closest complement are links. Darkest is background.<br />
Embed stylesheets in _includes as base.html as stylesheet for markdown post.<br />
Turn on Eleventy server.<br />
